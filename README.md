# Hippocam-camo
Hexagonal camouflage patterns for free use - code available under Creative Commons Attribution-Non-Commercial-ShareAlike 
https://creativecommons.org/licenses/by-nc-sa/4.0/

This file contains a script that generates different kinds of hexagonal camouflage patterns that have a close resemblance to how the Finnish military Camouflage M05 looks like.

If you don't feel like running Python scripts and experimenting, feel free to download .svg files with typical color palettes from [here](https://drive.google.com/drive/u/1/folders/1T3ZgYW78yHUQOClriGRgmsDusIVczJ3t). They are subject to the same Creative Commons License.
